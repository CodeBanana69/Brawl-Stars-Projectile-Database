# Brawl Stars Projectile Database

PNG sprite frame sequences for Brawl Stars projectile effects (attacks, gadgets, supers, and variants).

## Structure

```
assets/
  effects_brawler_<brawler>_<skin>_<ability>_projectile/
    frame_0000.png
    frame_0001.png
    ...
```

Each folder under `assets/` is one projectile effect. Frames are numbered sequentially for animation playback.
